# Desafío Grid - Punto Digital

Proyecto desarrollado para el bootcamp Full Stack JavaScript de Desafío Latam.

## 📚 Módulo
Módulo 2: CSS Avanzado

## 🧩 Unidad
Grid Layout

## 🚀 Descripción

Este desafío consiste en construir una página web tipo e-commerce utilizando CSS Grid para organizar la estructura principal del sitio.

El proyecto incluye:

- Navbar con imagen de fondo
- Menú lateral de categorías
- Grilla de productos
- Footer
- Diseño responsive básico
- Uso de imágenes y cards de productos

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- CSS Grid
- Flexbox

## 🎯 Objetivos del desafío

- Comprender el funcionamiento de CSS Grid
- Organizar layouts mediante `grid-template-areas`
- Trabajar con filas y columnas
- Aplicar estilos responsivos
- Integrar Grid y Flexbox en un mismo proyecto
